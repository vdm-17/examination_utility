# Программа-экзаменатор
Программа, парсящая пользовательскую библиотеку из файлов формата markdown, содержащих списки вопросов и ответов к ним, после чего в выбранном порядке демонстрирующая их пользователю, чтобы тот мог проверить и закрепить свои знания.

## Использование
1. Убедитесь, что в вашей операционной системе установлен python версии *>= 3.0.0*, и путь к интерпретатору содержится в переменной окружения *PATH*.

2. Скачайте zip-архив со [cтраницы программы в github](https://github.com/vdm-17/examination_utility).

3. Распакуйте содержимое архива по пути, который не содержит кириллицу или спец. символы.

4. Создайте каталог библиотеки с файлами формата markdown внутри, содержащими списки с вопросами, составленными в соответствие с [описанным ниже форматом](#формат-файлов-со-списками-вопросов).

5. Установите необходимые переменные окружения в соответсвие с [инструкцией ниже](#установка-переменных-окружения).

6. Отредактируйте файл *config.ini* в соответствие с [инструкцией ниже](#конфигурация-программы).

7. Установите необходимые зависимости при помощи команды:
 ```pip install -r requirements.txt```.

8. Запустите файл *examination_utility.bat*, либо самостоятельно перейдите в каталог программы и введите команду: ```python main.py```.

## Формат файлов со списками вопросов
Простейший файл со списком вопросов выглядит так:
```
# Вопросы для самопроверки
1. Вопрос1. ||Ответ к вопросу 1.||
```
Список заголовков, с которых могут начинаться блоки с вопросами, а также символы, в которые ответы к вопросам должны заключаться всегда, указваются [в конфиге](#конфигурация-программы).

## Установка переменных окружения

Создайте файл *.env* в основном каталоге программы и отредактируйте его следующим образом:
```
APP_ENV='ТЕКУЩЕЕ_ОКРУЖЕНИЕ'
LIBRARY_DIR_PATH='ВАШ_ПУТЬ_К_КАТАЛОГУ_БИБЛИОТЕКИ'
OPENAI_BASE_URL='БАЗОВЫЙ_URL_ИСПОЛЬЗУЕМОГО_OPENAI_API'
OPENAI_API_KEY='ВАШ_OPENAI_API_КЛЮЧ'
```
где:

* *APP_ENV* (опционально) - текущее окружение программы. Может принимать значение *development* при разработке.

* *LIBRARY_DIR_PATH* (обязательно) - путь к вашему каталогу библиотеки в системе.

* *OPENAI_BASE_URL* (опционально) - базовый URL вашего OpenAI API.

* *OPENAI_API_KEY* (опционально) - ваш OpenAI API ключ. Указывается для работы в режиме экзамена.

## Конфигурация программы
1. Откройте файл *config.ini*.

2. В разделе *\[DEFAULT\]* отредактируйте значение ключа *SEARCHABLE_QUESTIONS_BLOCKS_HEADINGS_TEXT*, заменив его на список из блоков текста, каждый из которых указывается с новой строки, где каждый блок текста программа будет искать в заголовках, под которыми должны будут храниться списки с вопросами. По умолчанию используются два заголовка: *Вопросы для самопроверки* и *Вопросы для самоповтора*.

3. В разделе *\[DEFAULT\]* отредактируйте значение ключа *SYMBOLS_CONTAINING_ANSWERS*, заменив его на символы, в которые будут заключаться ответы на соответсвующие вопросы. По умолчанию используются символы *||*.

4. В разделе *\[EXAMINER_AGENTS\]* отредактируйте значение ключа *COMPARING_AGENT_OPENAI_MODEL*, заменив его на необходимую модель OpenAI для сравнения ответа пользователя с указанным в библиотеке правильным ответом. По умолчанию используется значение *gpt-5-mini*.

5. В разделе *\[EXAMINER_AGENTS\]* отредактируйте значение ключа *COMPARING_AGENT_REASONING_EFFORT*, заменив его на необходимое значение усилиия обдумывания модели OpenAI для сравнения ответа пользователя с указанным в библиотеке правильным ответом. По умолчанию используется значение *low*.

6. В разделе *\[EXAMINER_AGENTS\]* отредактируйте значение ключа *COMPARING_AGENT_VERBOSITY*, заменив его на необходимое значение разговорчивости модели OpenAI для сравнения ответа пользователя с указанным в библиотеке правильным ответом. По умолчанию используется значение *low*.

7. В разделе *\[EXAMINER_AGENTS\]* отредактируйте значение ключа *ESTMATING_AGENT_OPENAI_MODEL*, заменив его на необходимую модель OpenAI для поиска правильного ответа и сравнения его с ответом пользователя. По умолчанию используется значение *gpt-5.1*.

8. В разделе *\[EXAMINER_AGENTS\]* отредактируйте значение ключа *ESTIMATING_AGENT_REASONING_EFFORT*, заменив его на необходимое значение усилиия обдумывания модели OpenAI для поиска правильного ответа и сравнения его с ответом пользователя. По умолчанию используется значение *medium*.

9. В разделе *\[EXAMINER_AGENTS\]* отредактируйте значение ключа *ESTIMATING_AGENT_VERBOSITY*, заменив его на необходимое значение разговорчивости модели OpenAI для поиска правильного ответа и сравнения его с ответом пользователя. По умолчанию используется значение *low*.

10. В разделе *\[HINTING_AGENTS\]* отредактируйте значение ключа *SIMPLE_HINTING_AGENT_OPENAI_MODEL*, заменив его на необходимую модель OpenAI для генерации подсказки к правильному ответу на вопрос, указанному в библиотеке. По умолчанию используется значение *gpt-5-mini*.

11. В разделе *\[HINTING_AGENTS\]* отредактируйте значение ключа *SIMPLE_HINTING_AGENT_REASONING_EFFORT*, заменив его на необходимое значение усилиия обдумывания модели OpenAI для генерации подсказки к правильному ответу на вопрос, указанному в библиотеке. По умолчанию используется значение *low*.

12. В разделе *\[HINTING_AGENTS\]* отредактируйте значение ключа *SIMPLE_HINTING_AGENT_VERBOSITY*, заменив его на необходимое значение разговорчивости модели OpenAI для генерации подсказки к правильному ответу на вопрос, указанному в библиотеке. По умолчанию используется значение *low*.

13. В разделе *\[HINTING_AGENTS\]* отредактируйте значение ключа *SMART_HINTING_AGENT_OPENAI_MODEL*, заменив его на необходимую модель OpenAI для поиска правильного ответа на вопрос и генерации подсказки к нему. По умолчанию используется значение *gpt-5.1*.

14. В разделе *\[HINTING_AGENTS\]* отредактируйте значение ключа *SMART_HINTING_AGENT_REASONING_EFFORT*, заменив его на необходимое значение усилиия обдумывания модели OpenAI для поиска правильного ответа на вопрос и генерации подсказки к нему. По умолчанию используется значение *high*.

15. В разделе *\[SMART_HINTING_AGENTS\]* отредактируйте значение ключа *ESTIMATING_AGENT_VERBOSITY*, заменив его на необходимое значение разговорчивости модели OpenAI для поиска правильного ответа на вопрос и генерации подсказки к нему. По умолчанию используется значение *low*.
